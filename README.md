## 🛠️ Como Usar

Se quiser rodar algum exercício localmente:

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/exercism.git
   cd exercism
   ```

2. Navegue até o exercício desejado:
   ```bash
   cd python/hello-world
   ```

3. Execute o arquivo ou os testes conforme a linguagem:
   ```bash
   # Python
   python hello_world.py

   # Para rodar os testes
   pytest hello_world_test.py
   ```
